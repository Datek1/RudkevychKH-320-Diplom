    API

    Наш сайт фільмів використовує MovieDB API для отримання даних про фільми. Щоб скористатись API MovieDB, слідуйте наступним крокам:

    Крок 1: Отримання ключа API

    1. Зареєструйтесь на MovieDB і увійдіть до свого облікового запису.

    2. У налаштуваннях облікового запису перейдіть на вкладку "API" і створіть новий ключ API. Ви отримаєте доступ до ключа API, який буде використовуватись для запитів до MovieDB API.

    Крок 2: Використання API

    Ви можете використовувати наступні ендпоінти MovieDB API для отримання даних про фільми:

    Отримання популярних фільмів

    • URL: https://api.themoviedb.org/3/movie/popular?api_key=YOUR_API_KEY
    • Метод: GET
    • Опис: Цей ендпоінт повертає список популярних фільмів.

    Отримання деталей фільму

    • URL: https://api.themoviedb.org/3/movie/{movie_id}?api_key=YOUR_API_KEY
    • Метод: GET
    • Опис: Цей ендпоінт повертає деталі конкретного фільму за його ідентифікатором movie_id.

    Пошук за жанром

    • URL: /api/movies/genre/{genreId}
    • Метод: GET
    • Опис: Цей ендпоінт повертає список фільмів за заданим жанром. Жанр вказується за його ідентифікатором genreId.

    Запит на жанри

    • URL: /api/genres
    • Метод: GET
    • Опис: Цей ендпоінт повертає список доступних жанрів фільмів.

    Пошук фільмів

    • URL: https://api.themoviedb.org/3/search/movie?api_key=YOUR_API_KEY&query=SEARCH_QUERY
    • Метод: GET
    • Опис: Цей ендпоінт повертає результати пошуку фільмів за заданим запитом SEARCH_QUERY.

    Замість YOUR_API_KEY підставте отриманий ключ API зі свого облікового запису MovieDB.

    Зауваження

    Переконайтеся, що ви дотримуєтесь умов використання MovieDB API та обмежень, які встановлені на їхньому сайті. Крім того, рекомендується зберігати ключ API в захищеному місці і не розголошувати його публічно.

    Для детальної документації та додаткових ендпоінтів MovieDB API звертайтеся до офіційного документації MovieDB API.
