    Вітаємо на нашому сайті фільмів!

    Цей веб-сайт пропонує вам переглядати інформацію про фільми, шукати нові релізи та додавати їх до свого списку перегляду. Ми використовуємо ряд потужних технологій та бібліотек для забезпечення високої якості користувацького досвіду.

    Основні функції

    • Переглядайте список фільмів з різних жанрів та категорій.
    • Пошук фільмів за назвою, режисером або актором.
    • Додавайте фільми до свого списку "Переглянути пізніше".
    • Оцінюйте та залишайте відгуки про фільми, які вам сподобалися.
    • Отримуйте рекомендації на основі ваших попередніх вподобань.

    Використані технології

    • React: Ми використовуємо React для побудови ефективного та динамічного інтерфейсу користувача.
    • Axios: Для здійснення запитів до API ми використовуємо Axios для зручної та надійної комунікації з сервером.
    • React Hook Form: Для управління формами нашого сайту ми використовуємо React Hook Form, що дозволяє нам легко збирати та перевіряти введені дані.
    • Redux Toolkit: Для управління станом даних ми використовуємо Redux Toolkit, який надає нам потужні засоби для організації та управління станом додатку.
    • TypeScript: Ми використовуємо TypeScript для покращення роботи з JavaScript, надаючи типи та забезпечуючи більшу безпеку в процесі розробки.
    • React DOM: Пакет для роботи з DOM в React-додатках.
    • React Slick: Бібліотека для створення каруселей (слайдерів) на основі React.
    • React Router DOM: Бібліотека для навігації та маршрутизації у React-додатках.
    • Slick Carousel: Бібліотека для створення стильних каруселей (слайдерів) на веб-сайтах.
